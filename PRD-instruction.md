# Moonlight/Sunshine 오픈소스 기반 학내 고성능 원격 컴퓨팅 플랫폼 구축

# 문제 정의
* 기존 운영 방식의 한계: 학내에 물리적인 고성능 데스크톱들을 배치하고 사용자가 직접 원격 접속하는 방식을 사용 중이나, 아래와 같은 문제들이 있음
    - 입력 지연(Latency): 기존 원격 제어 방식은 네트워크 환경에 따라 입력 반응이 늦어, 정밀한 조작이 필요한 고사양 소프트웨어 활용 시 작업 효율이 크게 저하됨
    - 예약 사용 기능 부재: 선착순 접속 방식의 한계로 인해 과제 마감이나 시험 기간 등 특정 시기에 자원을 확보하지 못해 업무와 학습이 중단되는 상황이 빈번함

# 타깃 사용자 및 사용 사례
* 핵심 사용자: 고사양 소프트웨어(SPSS, SAS, Adobe Suite, 3D 렌더링 툴 등) 활용이 필수적이나, 개인 기기(노트북, 태블릿 등)의 사양이 낮거나 라이선스 비용 문제로 인해 학내 고성능 인프라에 의존해야 하는 학내 구성원
* 사용 사례
    - 고부하 데이터 및 그래픽 작업: 개인용 노트북에서 구동하기 힘든 대용량 통계 분석, 영상 편집, 3D 모델링 작업을 학내 고성능 데스크톱 자원을 활용해 수행할 때
    - 라이선스 소프트웨어 원격 이용: 외부(자택, 카페 등)에서 고가의 학내 공용 소프트웨어를 별도 구매나 설치 없이 원격으로 접속하여 활용할 때

# 제안 해결책 (기술 전략: Moonlight & Sunshine)
* 초저지연 스트리밍 환경 구축
    - Sunshine(Host): 학내 고성능 데스크톱에 설치하여 GPU 하드웨어 가속 기반의 초저지연 인코딩을 수행, 실제 PC와 유사한 반응 속도 확보
    - Moonlight(Client) 활용: 다양한 OS 환경에서 60fps 이상의 안정적인 작업 환경을 제공하는 전용 클라이언트를 통해 접속
* 중앙 집중식 자원 예약 시스템: 캘린더 인터페이스를 통해 사용 가능한 데스크톱 리스트를 확인하고, 예약된 사용자에게만 특정 Sunshine 호스트의 접속 권한을 부여하는 중계 서버(Broker) 운영
* 포털 SSO 연동: 충남대 포털 계정과 연동하여 별도 가입 없이 예약 현황을 관리하고 접속 토큰을 발급받는 통합 인증 환경 제공
* 커스텀 Moonlight 프로토콜 핸들러: 사용자가 웹에서 '접속' 버튼을 누르면 브라우저가 moonlight:// 형태의 커스텀 URL을 호출하여, 로컬에 설치된 Moonlight 앱에 IP와 인증 정보를 자동으로 전달하는 방식 적용
* 자동 페어링(Auto-Pairing) 모듈: Sunshine API를 활용하여 사용자가 직접 PIN을 입력하는 과정 없이, 중계 서버가 호스트(PC)와 클라이언트 간의 인증을 백그라운드에서 즉시 완료
 
# 목표 및 성공 지표
* 목표: 전용 클라이언트를 통한 '게임 스트리밍' 수준의 압도적 성능을 제공하여 하드웨어 제약 없는 보편적 교육/업무권 보장
* 성공 지표
    - 접속 편의성: 서비스 접속부터 실제 제어까지 사용자가 입력해야 하는 정보 값(IP, PIN 등)을 0개로 축소
    - 조작 반응성: 입력 지연 시간을 사용자가 체감하기 어려운 수준인 20ms~50ms 이내로 단축
    - 자원 점유 성공률: 예약 시스템 도입을 통해 피크 시간대 서비스 가용성을 기존 대비 40% 이상 향상
    - 인프라 활용 효율: 예약 데이터를 분석하여 특정 PC 집중 현상을 방지하고 전체 기기의 가동률을 균등화

# MVP 요구사항
* 포털 연동 예약 페이지: 학교 계정으로 로그인하여 비어있는 고성능 PC와 사용 시간대를 선택하여 예약하는 기능
    - 충남대 포털에 로그인 된 상황에서는 본 시스템에 추가적인 로그인 불필요
* 동적 접속 토큰 발급: 예약이 승인된 사용자에 한해, 해당 시간 동안만 유효한 Sunshine 접속 PIN 또는 보안 토큰을 자동으로 생성하여 제공.
* 세션 관리 및 자동 차단: 예약 종료 10분 전 사용자 화면에 알림을 송출하고, 예약 시간이 종료되면 스트리밍 세션을 강제 종료하여 다음 예약자의 자원을 보장하는 기능.
* 호스트 상태 모니터링: 관리자가 현재 물리 데스크톱들의 연결 상태 및 부하 정도를 실시간으로 확인할 수 있는 대시보드
* 원클릭 접속 버튼: 예약 페이지 내 가용 PC 리스트에서 버튼 클릭 시 Moonlight 앱이 자동 실행되며 연결 시작
* 백그라운드 인증 연동: Sunshine API를 통해 호스트 PC와 사용자 기기 간의 PIN 인증 과정을 서버 단에서 자동 처리
* 실시간 PC 상태 체커: 고성능 데스크톱의 전원 상태 및 현재 사용 여부를 감지하여 '접속 가능'한 리스트만 동적으로 노출
* 세션 강제 종료 및 초기화: 예약 종료 시 세션을 자동으로 끊고, 다음 사용자를 위해 윈도우 로그아웃이나 임시 파일 삭제 스크립트 실행
